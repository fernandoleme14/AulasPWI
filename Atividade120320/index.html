SUA PÁGINA HTML; como estruturá-la?

De antemão, é preciso que você saiba as finalidades das principais TAGS.

Que são:

TAGS Para quê serve?
ARTICULANDO:
!-- Cria um comentário.
html Envolve todo o documento HTML.
head Envolve o cabeçalho de um documento HTML.
body Envolve o corpo (texto e tags) de um documento HTML.
TEXTO:
tittle Entitula o documento.
P Adiciona um parágrafo ao documento.
strong Escrita enfatizada em negrito.
I Escrita Itálica.
U Escrita sublinhada.
sub Subscrita.
sup Sobrescrita.
big Escrita maximizada.
LISTAGENS:
ul Lista em tópicos.
ol Lista numerada.
li Destaque a um item de lista.
menu Um menu com uma lista de itens.

Praticando:

Dadas as TAGS , deve-se notabilizar as; html, head, e body, que como TAGS de ARTICULAÇÃO, sendo fixas e pilarizando a estrutura html , é necessário (pelo menos aos iniciantes) que sejam precedentes de qualquer outra, assim, melhor organizando a sua página. Após atender a esse adendo, há um gigantesco leque de possibilidades para o desenvolvimento, prosseguimento e efetuação do que é idealizado, com milhares de outras tags.

Além das tags de articulação, são notáveis algumas das tags que realçam e acentuam a estética interfacial da página HTML. Tais como; style, id e class, tags CSS(sigla traduzida como "Folha de Estilo em Cascatas", no Português) que designam a modificação da estrutura html, a estilizando.

Atribuições:

    style

Contém informações de estilo para um documento ou uma parte do documento. As informações de estilo específico estão contidas dentro deste elemento, e podem ser atribuídas da inserção tanto dentro de "head", quanto "body".

    id

São uma forma de identificar um elemento, e devem ser ÚNICAS para cada elemento. É como se fossem impressões digitais de nossos dedos ou RGs. Através delas, pode-se atribuir formatação a um elemento em especial.

    class

As classes são uma forma de identificar um grupo de elementos. Através delas, pode-se atribuir formatação a VÁRIOS elementos de uma vez.
